Help
This application was designed as a simple schedule for FrOSCon. It downloads the schedule from FrOSCon server and saves it to your device. So you can use the schedule offline. You can also set your favorite schedule items to plan your conference visit.
Features:
  • Stores schedule. Use the "Refresh"-button in the lower left corner to download the latest schedule and automatically save it for offline use. The same also happens if you hit "Yes" after first launch.
  • Event details. Tap on an event to view detailed information.
  • Favorites. On the bottom of the event details you can set an event as favorite. Favorites are marked with a heart. In the menu on the bottom you can toggle between all events and your favorite events. This feature should help you with organizing your conference visit and remain focused.
  • Quick search. Simply start typing to search in all titles, locations and attendees.
  • Campus map. Use the campus map for orientation. You can find it in the application menu in the schedule scene. It's the one I found on the FrOSCon website.
  • Supports every orientation. Turn the device as you like to.
